HTML5

O que é HTML?

A HTML (HyperText Markup Language) é uma linguagem de marcação padrão para criação de sites. Ela é responsável por definir a estrutura de páginas web através de elementos diversos. Esses elementos são como rótulos para as diferentes partes de uma página, e determinam a forma de apresentação do conteúdo pelo navegador.

Elementos HTML

Um elemento HTML é composto por tag de abertura, conteúdo e tag de fechamento. A tag de fechamento diferencia-se pelo uso da barra (/).

<nomedatag> O conteúdo vai aqui... </nomedatag>

Alguns elementos podem até ser exibidos corretamente sem o udo da tag de fechamento, porém,é recomendado evitar tal prática. Outra recomendação é o uso de letras minúsculas, apesar do HTML não ser case sensitive.

Os elementos HTML podem ser aninhados, isto é, alguns elementos podem conter outros elementos.

<!DOCTYPE html>
  <html>
      <head>
        <title>Exemplo de Aninhamento</title>
      </head>
      <body>
      </body>
  </html>
  • <!DOCTYPE html> declara que o documento é um documento HTML5
  • <html> elemento raiz de uma página HTML
  • <head> contém meta informações sobre a página HTML, isto é, conteúdo não vísivel
  • <title> especifica um título para a página HTML (mostrado na barra de título do navegador ou na guia da página)
  • <body> define o corpo do documento e é um contêiner para todo o conteúdo visível

Tags HTML

São palavras reservadas utilizadas entres colchetes angulares (< e >). A maioria das tags possui uma abertura e um fechamento. Mas, existem os chamados elementos vazios, representados por tags que não possuem conteúdo e que são autofechadas. As tags podem ter atributos ou parâmetros, os quais podem receber diferentes valores responsáveis por comportamentos diversos.

Navegadores Web

Os navegadores têm a função de interpretar o documento HTML e apresentar seu conteúdo corretamente. Isso é possível mediante o uso adequado das tags, que não são exibidas em si, mas ,configuram a exibição de toda a página.

Editores HTML

A criação e modificação de páginas web podem ser feitas através de editores HTML profissionais ou até mesmo através de simples editores de texto. Para visualizar o documento HTML, deve-se salvar o arquivo com a extensão .html, ex: index.html

Estutura Básica

A estrutura atualmente proposta de um documento HTML básico é a seguinte:

<!DOCTYPE html>
<html lang="pt-br">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width,
    initial-scale=1.0">
    <title>Document</title>
  </head>
  <body>

  </body>
</html>

Todos os documentos HTML devem ser iniciados pela declaração <!DOCTYPE html>. Esta declaração aparece uma só vez e antes de quaisquer outras tags, representando o tipo de documento a ser exibido pelo navegador. O documento HTML tem seu início e fim demarcados pelo uso das tags <html>...</html>;. O atributo lang da tag html sinaliza a linguagem e região de origem do documento.
